✦ Frosted Glass

Every surface uses backdrop-filter with translucent RGBA layers. Cards, dialogs, navs — everything floats like glass panels.

🫧 Depth & Vibrancy

Layered translucency, glass borders, and multi-layer shadows create real visual depth — inspired by visionOS and macOS Sonoma.

📐 Semantic & Classless

Style <button>, <dialog>, <table> directly. Keto targets native elements and data-* attributes.

🎨 Variable Theming

Override --k-* CSS tokens and the entire library re-skins. Ships with 4 built-in theme presets.

⚡ Two-File Drop-In

One <link> + one <script defer>. Works from CDN, local, or npm. No bundler required.

♿ Accessible

ARIA roles, focus management, keyboard navigation, and prefers-color-scheme baked into every component.

A glass-era component library — under 15 KB, zero dependencies, Apple-inspired aesthetics.

How does glass styling work?

Keto uses backdrop-filter: blur() and RGBA surfaces to create frosted glass panels.

Does it need a build step?

Create your own theme

Pick any accent color and build a full theme in 3 steps. You only need to override the tokens you want to change — everything else inherits the defaults.

Step 1 — Choose an accent color

Set --k-primary and its hover/active/light variants. The -light variant should be your accent at ~10% opacity.

:root {
  --k-primary: #7c3aed;
  --k-primary-hover: #6d28d9;
  --k-primary-active: #5b21b6;
  --k-primary-fg: #ffffff;
  --k-primary-light: rgba(124, 58, 237, 0.1);
}

Step 2 — Set the surface palette

Adjust the background, text, and border colors to complement your accent. For light themes, tint --k-bg toward your accent. For dark themes, use deep dark values and invert glass opacities.

:root {
  /* Light purple theme */
  --k-bg: #faf5ff;
  --k-surface: rgba(255, 255, 255, 0.75);
  --k-text: #2e1065;
  --k-text-sec: #6b21a8;
  --k-text-ter: #a78bfa;
  --k-border: rgba(124, 58, 237, 0.1);
}

Step 3 — Tune the glass layer

The glass tokens control every translucent panel — cards, dialogs, dropdowns, tooltips, and the navbar. Adjust opacity and tint to match your palette.

:root {
  --k-glass-bg: rgba(255, 255, 255, 0.7);
  --k-glass-bg-heavy: rgba(255, 255, 255, 0.88);
  --k-glass-border: rgba(124, 58, 237, 0.12);
  --k-glass-shadow: 0 8px 32px rgba(124, 58, 237, 0.06);
}

Extend a preset at runtime

Use JavaScript to switch or extend themes dynamically. Override any token on document.documentElement.style:

// Apply a full theme at runtime
function setTheme(vars) {
  const root = document.documentElement.style;
  Object.entries(vars).forEach(([k, v]) => root.setProperty(k, v));
}

// Example: switch to a Sunset theme
setTheme({
  '--k-primary': '#f97316',
  '--k-primary-hover': '#ea580c',
  '--k-primary-light': 'rgba(249, 115, 22, 0.1)',
  '--k-bg': '#fff7ed',
  '--k-text': '#431407',
  '--k-glass-bg': 'rgba(255, 255, 255, 0.72)',
});

// Reset back to defaults
function resetTheme(keys) {
  keys.forEach(k => document.documentElement.style.removeProperty(k));
}

// Extend an existing theme — just add more overrides
setTheme({ '--k-rad': '16px', '--k-rad-lg': '22px' });

Scoped themes

CSS variable inheritance means you can theme any section independently. Just scope your overrides to a container class:

.dark-panel {
  --k-primary: #818cf8;
  --k-bg: #0f0e1a;
  --k-text: #f5f5f7;
  --k-text-sec: #a1a1a6;
  --k-glass-bg: rgba(22, 20, 38, 0.65);
  --k-glass-border: rgba(255, 255, 255, 0.1);

  background: var(--k-bg);
  color: var(--k-text);
  padding: 2rem;
  border-radius: var(--k-rad-xl);
}

/* All Keto components inside .dark-panel
   automatically pick up the scoped tokens */

Dark mode

Keto ships with a full dark palette. It automatically respects prefers-color-scheme: dark. You can also force a mode:

<!-- Auto — follows system preference (default) -->
<html>

<!-- Force dark -->
<html data-theme="dark">

<!-- Force light (even if user's system is dark) -->
<html data-theme="light">
